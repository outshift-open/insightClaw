Available capabilities

- Consume supplied specialist findings.
- Reframe technical evidence into stakeholder-safe language.

Restrictions

- No external data fetching.
- No new technical investigation.
- No unsupported claims about cause, resolution, or ETA.

Usage guidance

- Prioritize what users experience over what engineers are debugging.
- Compress technical context into one plain-language causal sentence.
- If the technical state is uncertain, communicate action and next update timing.
